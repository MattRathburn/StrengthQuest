# StrengthQuest

## Description

Strength Quest is a web application to track and monitor your progess using the Wendler 5/3/1 Strength training program.  You can add your Maxes to each lift and it will calculate the working max and set your weight and reps.

For information or to learn about the strength program visit [here](https://www.muscleandstrength.com/workouts/hardcore-look-at-jim-wendlers-5-3-1-powerlifting-system.html).

StrengthQuest utilizes Asp.Net Core 3.0, MsSql Server, Entity Framework Core 3.0 and Razor pages.  There is also a branch that is going to utilize Angular.
